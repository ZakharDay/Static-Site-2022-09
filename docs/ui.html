Название

Дизайн информационной системы: идея и устройство информационной системы

Описание

Студенты разрабатывают концепцию информационной системы, изучают целевую аудиторию и верстают одностраничный сайт с описанием проекта. Описывают объектную структуру проекта. Создают интерактивный прототип медиа-сервиса в Figma для формирования видения развития проекта. Изучают методология JTBD и учатся создавать Job Stories и User Stories.

Итоговый проект

Концепция сайта (информационной системы), макеты первичных артефактов, интерактивный прототип по методологии JTBD, схема структуры сайта.

Требования к промежуточному проекту
